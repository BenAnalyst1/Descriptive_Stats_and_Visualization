# Descriptive Analysis of Pupil Test Score Data
*Exploratory statistics and visualizations using RStudio*

## Objective
- To explore how key variables in the dataset relate to pupils' average test scores, using descriiptive statistics and visualizations.

## Tools & Packages
`R/RStudio` . `ggplot2` . `esquisse` . `Quarto`

## Approach
- Graphs were built using `esquisse`, a drag-and-drop interface from the `esquisse` package that generates `ggplot2` code automatically. This reduced time spent on manually building `ggplot2` codes and graphs and kept the workflow accessible while ensuring the code is fully reproducible. The analysis covers summary statistics, histograms, and scatter plots for the key variables. Each visual is accompanied by a brief written interpretation.
